A lightweight Chrome extension that lets you **copy the full transcript of a Coursera lecture with one click**.

Instead of manually selecting and copying transcript text, this extension extracts all transcript phrases automatically and copies them to your clipboard.


## How It Works

The extension scans the transcript container on the Coursera lecture page and extracts text from transcript phrases.

It collects all transcript sentences and copies them directly to the clipboard.


## Installation (Developer Mode)

1. Clone the repository

```
git clone https://github.com/your-github-user-name/coursera-transcript-exporter
```

2. Open Chrome

3. Go to

```
chrome://extensions/
```

4. Enable **Developer Mode**

5. Click **Load Unpacked**

6. Select the extension folder


## Usage

1. Open a Coursera lecture page
2. Click the extension icon
3. The transcript will be copied automatically
